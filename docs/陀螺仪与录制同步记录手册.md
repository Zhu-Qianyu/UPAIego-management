# 陀螺仪与视频录制同步记录手册（板端 `ros2_web_bridge`）

本文说明如何依据 **野火《快速使用手册—基于 LubanCat-RK3576 系列板卡》** §18.8（MPU6050、`/dev/i2c-7`、从机地址 `0x68`、陀螺仪寄存器自 `GYRO_XOUT_H=0x43` 起），在 **板端应用** `board/ros2_web_bridge` 中实现：**ffmpeg 开始写入本段 MP4 录像的同时，启动 MPU6050 陀螺仪 CSV 记录**，并与该段录像 **同目录、同文件名 stem**，便于后期对齐。

> **说明**：陀螺仪与「随 Web 桥启动的录像」同属板端进程，实现位于 `board/ros2_web_bridge/`，**不在** `upaiego/` 双相机节点包内。若你仅用 `upaiego` 做 OpenCV 双路录屏且不用 Web Bridge，需在 `upaiego` 中自行集成或改为只跑 `ros2_web_bridge`。

---

## 1. 硬件与手册依据（野火 §18.8）

| 项目 | 说明 |
|------|------|
| 板卡 | LubanCat-RK3576 系列 |
| 传感器 | MPU6050（I²C） |
| 典型设备节点 | **`/dev/i2c-7`**（手册 `i2c7-m1`；请以 `ls /dev/i2c-*` 实测） |
| 从机地址 | **`0x68`** |
| 陀螺仪寄存器 | **`0x43` 起连续 6 字节**（X/Y/Z 高低字节） |
| 初始化序列 | 与手册示例 `i2c_mpu6050.c` 中 `mpu6050_init` 一致（`PWR_MGMT_1`、`SMPLRT_DIV`、`CONFIG`、`ACCEL_CONFIG`） |

验证硬件可用：

```bash
gcc i2c_mpu6050.c -o mpu6050   # 来自野火例程路径
sudo ./mpu6050 /dev/i2c-7
```

## 2. 本仓库实现位置与行为

| 路径 | 作用 |
|------|------|
| `board/ros2_web_bridge/ros2_web_bridge/gyro_recorder.py` | `smbus2` 读 MPU6050，后台线程写 CSV |
| `board/ros2_web_bridge/ros2_web_bridge/node.py` | `ffmpeg` 成功启动后，启动陀螺仪；节点退出或重启录像前停止线程 |
| `board/ros2_web_bridge/config/bridge_params.yaml` | `gyro_*` 参数默认值 |

**输出文件命名**（与单次 ffmpeg 会话一一对应）：

- 录像：`{recording_dir}/camera_h264_YYYYMMDD_HHMMSS.mp4`
- 陀螺仪：**`{recording_dir}/camera_h264_YYYYMMDD_HHMMSS_gyro.csv`**（与 mp4 **同 stem** 加后缀 `_gyro.csv`）

**启停时机**：

- **启动**：`_ensure_recording()` 中 `subprocess.Popen(ffmpeg…)` 成功后立即 `GyroRecorder.start()`。
- **停止**：`destroy_node()` 中先 `GyroRecorder.stop()`，再终止 ffmpeg；若 ffmpeg 异常退出后再次 `_ensure_recording()` 开启新一段录像，会先停旧陀螺仪再起新会话。

## 3. 依赖与权限

1. **Python**：`ros2_web_bridge` 的 `setup.py` 已声明依赖 **`smbus2`**；`colcon build` 后若运行环境未装，可 `pip install smbus2`（与 overlay Python 一致）。
2. **权限**：访问 `/dev/i2c-*` 需用户加入 **`i2c`** 组或使用具备权限的用户运行节点。
3. **总线号**：默认 `gyro_i2c_bus: 7`；硬件不同请改 YAML。

---

## 4. ROS2 参数（`web_bridge_node`）

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| `gyro_enable` | `true` | 是否启用陀螺仪记录 |
| `gyro_i2c_bus` | `7` | 对应 `/dev/i2c-7` |
| `gyro_sample_interval_sec` | `0.01` | 采样间隔（秒） |

在 `bridge_params.yaml` 或 launch 中覆盖即可。

---

## 5. `*_gyro.csv` 格式

- 表头：`mono_ts_sec,gyro_x,gyro_y,gyro_z`
- `mono_ts_sec`：线程内单调时钟相对起点（秒）
- `gyro_*`：寄存器原始有符号整型；角速度换算需查 MPU6050 数据手册与量程位

---

## 6. 故障排查

| 现象 | 处理 |
|------|------|
| 日志 `smbus2 not installed` | 安装 `smbus2` 并确认运行该节点的 Python 环境 |
| `cannot open I2C bus` | 检查总线号、`i2c` 组、设备是否存在 |
| `MPU6050 init failed` | 接线、地址、是否被其他进程占用 |
| 仅有 mp4 无 csv | `gyro_enable: false` 或陀螺仪 `start()` 失败（见日志） |
